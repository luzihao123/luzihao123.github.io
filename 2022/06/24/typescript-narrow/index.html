typescript – 类型收窄

typeof 的类型

“number”
“bigint”
“boolean”
“symbol”
“undefined”
“object”

是否存在值判断

let a:number
if(a) {
    return a
}

类似这种写法，但是不能判断 0 和 空值之间的区别

运用 if条件判断 或者 in判断 做收窄

    type Fish = { swim: () => void };
    type Bird = { fly: () => void };
    type Human = { swim?: () => void; fly?: () => void };
 
    function move(animal: Fish | Bird | Human) {
        if ("swim" in animal) {
            animal;
            // fish human
        } else {
            animal;
            // bird human
        }
    }

使用instanceof

用于检查是否为另一个构造函数的实例

使用is 创建 类型保护函数

    function isFish(pet: Fish | Bird): pet is Fish {
        return (pet as Fish).swim !== undefined;
    }

    if (isFish(pet)) {
        pet.swim()
    } else {
        pet.fly()
    }

原因: 如果一个联合类型包含两个类型，两个类型中不重合的属性的，使用时会报错，xx属性不属于某个类型， ts具有严格的空值判断！
如pet (Fish | Bird), 如果直接使用pet.swim, 是会报错的，这时候可以使用if语句配合 isFish()函数做类型收窄
当然也可以使用 ！做类型断言，或者直接把类型值都设为可选。

never 类型

一般可以把never类型放在类型判断的最后，用以做报错提醒，因为没有任何类型可以分配给never类型， 当never类型有值时就会报错
